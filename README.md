# Recursive-Descent-Compiler
Recursive-Descent Parsing Compiler – C/C++, Compilers, Programming Languages, Object Oriented Programming

• Developed a recursive-descent compiler for parsing, semantic analysis, and execution of programs defined by a custom grammar in C++11.

• Implemented semantic analysis and execution support for classic programming language features including variable declarations,
  arithmetic expressions, conditional statements, loops, and control flow, along with semantic error detection.
